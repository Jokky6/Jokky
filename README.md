## 启动服务

```
git clone https://git.coding.net/indexer/Lin-Doc.git // 拷贝代码
cd Lin-Doc    // 进入项目根目录
yarn          // 安装依赖
yarn dev      // 启动服务
```

## 更改配色

把`node_modules/prismjs/themes/`里的`prism-tomorrow.css`替换成根目录下的`prism-tomorrow.css`.
